# OLW Hiring Task – Airbnb Assistant Landing Page

This project is a fully responsive landing page built with **Next.js** for the **OneLittleWeb Web Developer Hiring Task**.  
The main goal was to convert the provided Figma design into a clean, accurate, and production-ready frontend implementation.
